# Ruby Fundamentals Documentation

This project contains foundational examples and explanations of Ruby syntax and features. It's structured in different folders, each representing a key concept or topic.

## 📁 Structure

- **calculator/**  
  A simple command-line calculator built using Ruby, showcasing methods, conditionals, and input handling.

- **conditional-loops/**  
  Examples demonstrating the use of `if`, `unless`, `case`, `while`, `until`, `for`, and `each` in Ruby.

- **data-types/**  
  Covers Ruby's core data types:
  - `main.rb` — Overview of all data types
  - `arrays.rb` — Array methods and usage
  - `numbers.rb` — Integer, float, and number operations
  - `strings.rb` — String manipulation and methods

- **handle-errors/**  
  Demonstrates Ruby’s error handling with:
  - `begin-rescue`
  - `ensure`
  - Custom exceptions
  - Error handling in methods

- **methods/**  
  Explains how to define and use methods in Ruby, including default parameters and variable-length argument lists (`*args`).

- **classes-objects**: Demonstrates Ruby's object-oriented programming (OOP) features, including defining classes, initializing objects, and using methods. It also covers `attr_accessor`, `attr_reader`, and `attr_writer`.

## Requirements

- Ruby installed on your system (version 2.5+ recommended)
- A text editor or IDE like VS Code, Sublime Text, or RubyMine
